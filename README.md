# RpiDormView

RpiDormView is an open source project designed to help prospective students, current students and parents learn more about dorms at RPI.


Tech Stack: HTML, Python, Typescript, CSS

Philosophy
==========

The main goals for this project are listed below.

**Find a dorm that fits your needs**

Filter dorms based on:
- Price
- Rooming Types
- Amenities
- Parking Nescessity
- And More!
